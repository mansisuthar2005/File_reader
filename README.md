# File_reader
This contains a code which can help you read and write in a text file or convert the content into a spoken audio.
This project provides a simple Python utility that can read, write, and convert text content into spoken audio using the pyttsx3 text-to-speech engine. It is designed to help automate text processing tasks while also enabling audio output for accessibility or convenience.
